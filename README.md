Window Switcher
===============
Change the active window gracefully from the keyboard.

![demo screenshot](demo/explanation.gif)

Usage
-------
1. Launch this program
2. Each window is labeled by a single letter
3. Type a key to active the associated window

Environment
------------
PyGTK
